AIS3 Day1

2019-07-29 / 1 min read

住宿

從今天開始，是 AIS3 2019 的第一天，其實昨天就入住交大十舍（男生）、竹軒（女生）。昨天在交誼廳看作業看到 0 點，發現 0 點宿舍就會熄燈（在此譴責交大十舍房間內沒有任何燈具…）
這邊先附上 共筆

早上

由逢甲資工的王銘宏教授講了一假新聞研究，不得不說真的還蠻酸的 XDD（有興趣可參看共筆），先是介紹了假新聞的傳播、內容之類的。後來提到假新聞的辨別方式。在最後用 R 分析 PTT 八卦版的資料。（但教授螢幕字超小 QQ ，完全看不到，之前也沒碰過 R ，整個直接爆炸）

下午

下午則是由中華電信的王凱慶專員帶來 WEB 漏洞，講師疑似鄭教授的學生（？），以前好像就有打 CTF ，也放了不少打 CTF 的人知道的梗。

每一次的操作，都會造成講者錢包的傷害

整個議程主要在 SSRF ，AWS 的部份還有 API endpoint 的揭露等等。

晚上

這是今年 AIS3 特別的地方，以往的 Final 好像是組隊打 CTF，但鄭欣明教授參考了韓國 GCC 的作法後，覺得資安不全是 CTF 的形式，因此改為專題分組報告。所以第一天晚上就是抽籤分組，以及彼此自我介紹。接下來要想想第六天的專題報告的題目（但我這組是軟體開發安全，但大家都不太清楚軟體開發安全的定義是什麼，助教也還沒出現，所以題目難產拉QQ）
